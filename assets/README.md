# Assets

Esta pasta contém os recursos visuais do aplicativo.

## Imagens necessárias:

### Ícones
- `icon.png` - 1024x1024px - Ícone principal do app
- `adaptive-icon.png` - 1024x1024px - Ícone adaptativo (Android)
- `favicon.png` - 48x48px - Favicon para web
- `splash.png` - 1242x2436px - Splash screen

### Geração de Ícones

Você pode usar ferramentas online gratuitas para gerar ícones:
- [App Icon Generator](https://www.appicon.co/)
- [MakeAppIcon](https://makeappicon.com/)
- [Expo Icon Generator](https://expo.github.io/tools/)

### Sugestões de Design

Para o ícone do ArsSétima, considere:
- Símbolo do número 7 estilizado
- Elementos místicos (estrelas, runas, símbolos arcanos)
- Paleta de cores do app (roxo #6B21A8, lavanda #9F7AEA)
- Estilo minimalista ou detalhado conforme preferência

### Como adicionar os ícones:

1. Crie ou baixe os ícones necessários
2. Coloque-os nesta pasta com os nomes exatos listados acima
3. O Expo automaticamente os reconhecerá

### Imagens de exemplo (seed data):

Coloque imagens de personagens e locais em:
- `assets/images/`

Formatos suportados: PNG, JPG, WEBP
